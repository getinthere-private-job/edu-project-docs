## 프로젝트 개요

| 항목 | 내용 |
|-----|------|
| 프로젝트명 | 1차 모놀로식 프로젝트 (자유주제) |
| 프로젝트 유형 | PBL(Project Based Learning) |
| 훈련시간 | 140시간 (약 4주) |
| 기술스택 | Spring Boot, MySQL, Mustache, Redis |
| 팀 구성 | 4~6명 / 팀 |

## PBL 단계별 운영

| 단계 | 기간 | 활동 | 산출물 |
|-----|------|------|--------|
| 1단계: 문제 인식 | 1~2일 | 팀 구성, 주제 선정 | 팀 구성표 |
| 2단계: 학습 계획 수립 | 3~5일 | PBL 프로젝트 계획서 작성, 설계 | PBL 프로젝트 계획서, ERD, API 설계서 |
| 3단계: 구현 (Sprint 1) | 6~10일 | 핵심 기능 구현, 자기주도 학습 | 핵심 기능, 하브루타노트 |
| 4단계: 중간 점검 | 11일 | 중간 발표, 멘토 피드백 | 중간 발표 자료 |
| 5단계: 고도화 (Sprint 2) | 12~17일 | 추가 기능, 코드리뷰 | 구현 코드, 코드리뷰 기록 |
| 6단계: 최종 발표 | 18~20일 | 최종 발표, KPT 회고 | 최종 발표 자료, 기술 블로그 |
