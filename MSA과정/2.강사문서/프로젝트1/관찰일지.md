## 대인관계 능력 평가 기준

| 점수 | 등급 | 기준 | 근거자료 |
|-----|------|------|---------|
| 5점 | 매우 우수 | 팀원 간 갈등을 주도적으로 조율, 모든 팀원의 의견을 적극 수렴 | 회의록 발언 기록, 갈등 조율 사례 |
| 4점 | 우수 | 팀 활동에 적극 참여, 다른 팀원을 돕는 모습 관찰 | 코드리뷰 참여, 페어 프로그래밍 |
| 3점 | 보통 | 주어진 역할을 성실히 수행하나 적극적 소통 부족 | Git 커밋, 회의 참석, 스탠드업 참여 |
| 2점 | 미흡 | 팀 활동 참여가 소극적, 소통 원활하지 않음 | 회의 불참, 커밋 빈도 저조 |
| 1점 | 매우 미흡 | 팀 활동에 거의 참여하지 않음 | 무단 불참, 갈등 발생 기록 |

## 문제해결 능력 평가 기준

| 점수 | 등급 | 기준 | 근거자료 |
|-----|------|------|---------|
| 5점 | 매우 우수 | 복잡한 기술 문제를 자기주도적으로 분석·해결, 미학습 기술도 독자 학습 적용 | 하브루타노트 학습 기록, 블로그, 코드리뷰 사례 |
| 4점 | 우수 | 기술 문제 발생 시 적극적으로 원인 분석, 다양한 접근 시도 | Git PR 기록, 하브루타노트 문제 해결 기록 |
| 3점 | 보통 | 기본 문제는 해결 가능, 복잡한 문제에서는 도움 필요 | 질의응답, 멘토링 요청, 기능 구현 목록 |
| 2점 | 미흡 | 문제 해결 시도 부족, 다른 팀원에게 의존 | 커밋 부족, 기능 미완료, 학습 기록 미비 |
| 1점 | 매우 미흡 | 문제 해결 의지 없음, 학습 노력 거의 없음 | 산출물 미제출, 학습 기록 없음 |
